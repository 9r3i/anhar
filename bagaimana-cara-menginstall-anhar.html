1. Apa itu Anhar?

 

Anhar diambil dari bahasa 'Arab yang berarti sungai. Namun Anhar disini adalah sebuah framework untuk aplikasi-aplikasi yang berbasis HTML. Didukung dengan CSS (Cascade Style Sheet) untuk mempercantik tampilan dan Javascript (Scripting Language) untuk menunjang aplikasi-aplikasi dan framework itu sendiri, yang memungkinkan Anhar untuk dapat memuat beberapa aplikasi dalam satu file saja, dengan metode offline, dan dapat diakses kapan saja tanpa menggunakan data internet.

 

2. Tujuan

 

Tujuan utama pembuatan framework ini adalah untuk mempermudah umat Islam memami agama. Oleh karena itu, aplikasi pertama yang tertanam pada framework ini adalah Al Quran versi terjemah bahasa Indonesia. Selanjutnya akan dikembangkan beberapa aplikasi yang menunjang dalam mempelajari agama Al Haqq, Al Islam.

9. Requirement-file

 

Adalah file-file yang dibutuhkan atau wajib disertakan pada directori yang sama dengan anhar.html pada sebuah aplikasi

Biasanya file tersebut adalah:

1. *.json, adalah file untuk informasi tentang aplikasi tersebut, [namespace].json, contoh: quran3id.json

2. *.js, adalah file untuk memuat aplikasi tersebut berbentuk Javascript, [namespace].js, contoh: quran3id.js

3. dan *.png, adalah file icon yang ditampilkan di beranda Anhar, contoh: anhar-64.png
